About This Project

Hello Spring Boot is a demonstration project showcasing modern Spring Boot application development and deployment practices. Built with Java 17 and Spring Boot 3.5.5, this application demonstrates containerization, database integration, automated testing, and cloud deployment strategies.

Technology Stack

Java 17 Spring Boot 3.5.5 Maven 3.9.9 PostgreSQL H2 Database Docker Render.com GitHub Actions

Getting Started

1

Clone Repository

Clone the repository to your local machine:

git clone https://github.com/Abhay2133/hello_spring.git
2

Run Application

Start the application using Maven wrapper:

./mvnw spring-boot:run
3

Access Endpoints

Visit the application endpoints:

GET / - Homepage
GET /health - Application health status
GET /memory - Memory usage information
GET /ping-status - Ping service status
